# SLcc

This is a compiler for Huawei Bisheng Cup.

## SLcc使用

* 打开docker

```shell
    make docker
```

* 清空编译结果

```shell
    make clean
```

* 自动测试

```shell
    make autotest
```

* 手动测试

```shell
    make TARGET=-riscv/-koopa test
```

* 编译koopa

```shell
    make koopa-test
```

* 编译riscv

```shell
    make riscv-test
```
